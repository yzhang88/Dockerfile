# Dockerfile
### 小组介绍  
`HelloBioinfo` 是生物信息学习交流小组，欢迎加入我们。
### 项目介绍
做生物信息工作经常遇到软件安装问题， `docker` 是个不错的解决方案，但是 `docker` 制作镜像有时候也很麻烦。这个项目存放本小组成员原创并测试可用的 `Dockerfile` 。

| --- | --- |
| [debian_stretch-R](./debian_stretch-R) | 基于 `debian-stretch` 的 `R` 语言，适合作为基础镜像安装R包。 |
